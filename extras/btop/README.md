## Usage

### Install

1. Place `cyberdream.theme` or `cyberdream-light.theme` under `~/.config/btop/themes/`

2. Open `btop`, press `Esc`, select **Options**, then choose **cyberdream** or **cyberdream-light** from the color theme list.

Alternatively, edit `~/.config/btop/btop.conf`:

```conf
color_theme = "cyberdream"
```

For more information, see the [btop documentation](https://github.com/aristocratos/btop).
