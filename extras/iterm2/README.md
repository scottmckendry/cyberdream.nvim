## Usage

### Install

1. Download `cyberdream.itermcolors` or `cyberdream-light.itermcolors`.
2. Double-click the file, or in iTerm2 go to **Settings → Profiles → Colors → Color Presets → Import…** and select the file.
3. Select the imported preset from the **Color Presets** dropdown.

For more information, see the [iTerm2 documentation](https://iterm2.com/documentation-preferences-profiles-colors.html).
