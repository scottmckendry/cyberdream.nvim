## Usage

### Install

1. Copy the theme file to vicinae's themes directory:

```bash
# Dark theme
cp cyberdream.toml ~/.local/share/vicinae/themes/

# Light theme
cp cyberdream-light.toml ~/.local/share/vicinae/themes/
```

2. Apply the theme using one of the following methods:

**Using the UI:**

- Open Vicinae's root search
- Type "Set Theme" and select the command
- Choose "Cyberdream" (dark) or "Cyberdream Light" from the list

**Using the CLI:**

```bash
# Dark theme
vicinae theme set cyberdream

# Light theme
vicinae theme set cyberdream-light
```

More info can be found in the [vicinae theming docs](https://docs.vicinae.com/theming/getting-started)
